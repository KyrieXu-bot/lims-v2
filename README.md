# LIMS (React + Express + MySQL + JWT)

Admin-only CRUD for **Customers / Payers / Commissioners** with scalable UI shell.

## Start backend
```
cd server
npm i
# edit .env
npm run dev
```

## Start frontend
```
cd client
npm i
npm run dev
```
Open http://localhost:5173 and login with ADMIN_USER / ADMIN_PASS from server/.env.
